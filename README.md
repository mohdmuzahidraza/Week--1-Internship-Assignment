# ☕ Java Mini Projects Collection

This repository contains three beginner-friendly Java console applications developed to strengthen core Java programming concepts such as user input handling, conditional statements, loops, and basic business logic.

---

📋 Projects Included

1. 🏦 Bank Management App

Overview
A simple banking application that allows users to perform basic banking operations through a menu-driven interface.

Features
- Deposit Money
- Withdraw Money
- Check Balance
- Input Validation
- Insufficient Balance Check
- Exit Option

Technologies Used
- Java
- Scanner Class
- Loops
- Conditional Statements

Sample Menu

1. Deposit
2. Withdraw
3. Check Balance
4. Exit

Learning Concepts
- While Loop
- If-Else Conditions
- User Input Handling
- Banking Logic

---

2. 🧮 Simple Calculator

Overview
A basic calculator application that performs arithmetic operations on two numbers entered by the user.

Features
- Addition
- Subtraction
- Multiplication
- Division
- Division by Zero Handling

Technologies Used
- Java
- Scanner Class
- Arithmetic Operators

Supported Operations

Addition       : a + b

Subtraction    : a - b

Multiplication : a * b

Division       : a / b

Learning Concepts
- Mathematical Operations
- Input Handling
- Conditional Logic
- Error Prevention

---

3. 🎓 Grade Evaluation System

Overview
A Java application that evaluates student grades based on marks entered by the user.

Features
- Accept Student Marks
- Automatic Grade Assignment
- Easy-to-Use Interface
- Instant Result Generation

Grading Criteria

| Marks | Grade |
|---------|---------|
| 80 - 100 | A |
| 60 - 79 | B |
| 40 - 59 | C |
| Below 40 | Fail |

Technologies Used
- Java
- Scanner Class
- If-Else Statements

Learning Concepts
- Decision Making
- Grade Calculation
- User Input Processing
